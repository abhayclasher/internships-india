# Contributors Guide

Thank you for your interest in contributing to this repository!

## How to Contribute

### 1. Add New Internships
If you find new internship opportunities, feel free to add them to the repository:

1. Fork the repository
2. Add the internship to the appropriate markdown file
3. Create a pull request

### 2. Report Issues
Found outdated information or broken links? Let us know by creating an issue.

### 3. Share Feedback
Have suggestions? We'd love to hear from you!

## Contribution Guidelines

- Keep information accurate and up-to-date
- Use proper formatting for markdown files
- Include direct apply links when possible
- Maintain the existing structure
